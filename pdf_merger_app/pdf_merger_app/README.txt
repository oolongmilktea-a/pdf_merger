PDF Merger App

This is a simple Python program that lets you merge two PDF files into one.

What is included:
- pdf_merger.py

Requirements:
- Python 3
- pypdf

Install dependency:
    pip install pypdf

How to run:
    python pdf_merger.py

How to use:
1. Click Browse next to First PDF and choose the first file.
2. Click Browse next to Second PDF and choose the second file.
3. Choose where to save the merged PDF.
4. Click Merge PDFs.

Notes:
- The pages from the first PDF are added first, then the pages from the second PDF.
- If you want, I can also make a version that merges any number of PDFs, not just two.
